Function Code

function confirmEnding(strToCheck, strToCheckAgainst) {
  
  let sizeOfAgainst = strToCheckAgainst.length;
  console.log(sizeOfAgainst);
  console.log(strToCheck.slice(-sizeOfAgainst));

  if (strToCheck.slice(-sizeOfAgainst) == strToCheckAgainst) {
    return true;
  }
  else return false;
}

console.log(confirmEnding("Connor", "nor")); // Example usage

How It Works

This function checks if the first string ends with the second string by:

1. Calculating the length of the second string

2. Extracting a substring from the end of the first string with the same length

3. Comparing this substring with the second string

4. Returning true if they match, false otherwise

  • Note: This is a strict String checker, whitespaces are checked too!
